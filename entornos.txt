
En Windows:
powershell
python -m venv venv_win
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt

En Linux:
bash

python3 -m venv venv_linux
. .venv/bin/activate
pip install -r requirements.txt

Ambos entornos se llaman venv, pero son independientes. Podés ignorar uno cuando estás en el otro sistema.
